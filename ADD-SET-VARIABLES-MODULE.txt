================================================================================
HOW TO ADD MODULE 8 (SET VARIABLES)
================================================================================

STEP 1: Add Module
------------------
1. Open Make.com
2. Open "Email 1 - Invitation" scenario
3. Click "+" button (add module)
4. Search for "Set Variables"
5. Select "Set Variables" module
6. It will be added to the flow

STEP 2: Position Module 8
--------------------------
1. Module 8 should be BETWEEN Module 1 and Module 2
2. Flow should be:
   Module 1 → Module 8 → Module 2

3. If Module 8 is in wrong position:
   - Drag it between Module 1 and Module 2
   - OR delete and re-add it after Module 1

STEP 3: Connect Modules
------------------------
1. Connect Module 1 to Module 8:
   - Drag from Module 1 output to Module 8 input
   - OR click Module 8 → Select Module 1 as input

2. Connect Module 8 to Module 2:
   - Drag from Module 8 output to Module 2 input
   - OR click Module 2 → Select Module 8 as input

STEP 4: Configure Module 8 Variables
------------------------------------
1. Click on Module 8 (Set Variables)
2. Add 3 variables:

Variable 1: name
   - Name: name
   - Value: {{1.0}}
   - (NOT {{1.`0`}} - no backticks!)

Variable 2: email
   - Name: email
   - Value: {{1.1}}
   - (NOT {{1.`1`}} - no backticks!)

Variable 3: language
   - Name: language
   - Value: {{1.3}}
   - (NOT {{1.`3`}} - no backticks!)

3. Set "Variable lifetime" or "Scope":
   - Select: "One cycle" or "Roundtrip"

4. Save Module 8

================================================================================
UPDATE MODULES TO USE MODULE 8 VARIABLES
================================================================================

After adding Module 8, update these modules:

MODULE 3 (English Email):
--------------------------
- Filter: {{8.language}} = "en" (instead of {{1.3}})
- To: [{"value": "{{8.email}}"}] (instead of {{1.1}})
- Subject: "{{8.name}}, You're Invited..." (instead of {{1.0}})
- HTML: {{8.name}} (instead of {{1.0}})

MODULE 4 (Arabic Email):
-------------------------
- Filter: {{8.language}} = "ar" (instead of {{1.3}})
- To: [{"value": "{{8.email}}"}] (instead of {{1.1}})
- Subject: "{{8.name}}، أنت مدعو!..." (instead of {{1.0}})
- HTML: {{8.name}} (instead of {{1.0}})

================================================================================
WHY USE MODULE 8?
================================================================================

ADVANTAGES:
-----------
1. Cleaner code: {{8.email}} is clearer than {{1.1}}
2. Easier debugging: Check Module 8 output to see what data was retrieved
3. Reusable: Can use {{8.email}} in multiple places
4. Better organization: All data in one place

DEBUGGING:
----------
When you check execution logs:
- Module 1: Shows raw Google Sheets data
- Module 8: Shows organized variables (name, email, language)
- Easy to see if data is correct!
