================================================================================
COMMON ISSUES AND FIXES
================================================================================

ISSUE 1: Connection Broken
--------------------------
SYMPTOM: Module 1 shows connection error

FIX:
1. Click on connection in Module 1
2. Re-authenticate
3. Grant all permissions
4. Save

ISSUE 2: Form ID Wrong
-----------------------
SYMPTOM: Module 1 can't find form

FIX:
1. Get correct Form ID from Google Forms
2. Update in Module 1
3. Save

ISSUE 3: Scenario Not ON
------------------------
SYMPTOM: No executions at all

FIX:
1. Turn scenario ON (green toggle)
2. Wait 30 seconds
3. Test again

ISSUE 4: Instant Mode Enabled
-----------------------------
SYMPTOM: Trigger not working properly

FIX:
1. Go to scenario settings
2. Turn OFF "Instant" mode
3. Save

ISSUE 5: No New Submissions
----------------------------
SYMPTOM: Form not being submitted

FIX:
1. Test form submission yourself
2. Verify form is accessible
3. Check form settings
