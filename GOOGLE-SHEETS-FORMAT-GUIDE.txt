REQUIRED COLUMN STRUCTURE
================================================================================

Your Google Sheets should have these columns (in this order):

| Column | Header        | Required | Example              | Notes                    |
|--------|---------------|----------|----------------------|--------------------------|
| A      | Name          | YES      | "John Smith"         | Lead's full name         |
| B      | Email         | YES      | "john@example.com"   | Valid email address      |
| C      | Phone         | NO       | "+968 12345678"      | Optional but recommended |
| D      | Language      | YES      | "en" or "ar"        | Must be "en" or "ar"      |
| E      | Status        | NO       | "New"               | Auto-updated by system   |
| F      | Date Added    | NO       | Auto-filled         | Auto-filled              |
| G      | Email 1 Date  | NO       | Auto-filled         | Auto-filled              |
| H      | Email 2 Date  | NO       | Auto-filled         | Auto-filled              |
| I      | Email 3 Date  | NO       | Auto-filled         | Auto-filled              |
| J      | Notes         | NO       | "Interested in..."   | Optional notes           |

================================================================================
LANGUAGE VALUES
================================================================================

IMPORTANT: Language column (D) must be EXACTLY:
- "en" (lowercase) for English
- "ar" (lowercase) for Arabic

OR:
- "EN" (uppercase) for English
- "AR" (uppercase) for Arabic

BUT: Make sure your router filters match the case you use!

RECOMMENDED: Use lowercase "en" and "ar" for consistency.

================================================================================
WHAT HAPPENS AFTER YOU ADD A LEAD
================================================================================

IMMEDIATELY (within seconds):
-----------------------------
1. Make.com detects new row
2. Reads the data:
   - Name: Column A
   - Email: Column B
   - Language: Column D
3. Routes based on Language
4. Sends Email 1 (English or Arabic)
5. Updates Google Sheets:
   - Status (E) = "Email 1 Sent"
   - Email 1 Date (G) = Current date/time

AFTER 3 DAYS:
-------------
1. Scenario 2 runs daily at 9:00 AM
2. Finds leads with:
   - Status = "Email 1 Sent"
   - Email 1 Date = More than 3 days ago
3. Sends Email 2
4. Updates:
   - Status = "Email 2 Sent"
   - Email 2 Date (H) = Current date/time

AFTER 4 MORE DAYS (7 days total):
-----------------------------------
1. Scenario 3 runs daily at 9:00 AM
2. Finds leads with:
   - Status = "Email 2 Sent"
3. Sends Email 3
4. Updates:
   - Status = "Email 3 Sent"
   - Email 3 Date (I) = Current date/time

BULK IMPORT FROM CSV/EXCEL
================================================================================

If you have many leads in a CSV or Excel file:

STEP 1: Prepare Your CSV/Excel
--------------------------------
Make sure it has these columns (in order):
- Column 1: Name
- Column 2: Email
- Column 3: Phone (optional)
- Column 4: Language ("en" or "ar")
- Column 5: Status (can leave empty)

STEP 2: Import to Google Sheets
--------------------------------
1. Open Google Sheets
2. File ‚Üí Import
3. Upload your CSV/Excel file
4. Make sure columns align correctly

STEP 3: Verify Format
----------------------
Check that:
- Column A = Name
- Column B = Email
- Column D = Language ("en" or "ar")

STEP 4: Save
------------
Save the sheet

STEP 5: Automatic!
-------------------
Make.com will detect all new rows
Email 1 will be sent to ALL new leads automatically!

================================================================================
TIPS FOR ADDING LEADS
================================================================================

TIP 1: Use Consistent Language Values
------------------------------------
Always use "en" or "ar" (lowercase) for consistency

TIP 2: Validate Email Addresses
--------------------------------
Make sure email addresses are valid before adding
Invalid emails will cause errors

TIP 3: Add Phone Numbers
------------------------
While optional, phone numbers are useful for follow-up

TIP 4: Use Status Column
-------------------------
You can manually set Status to skip emails:
- Set to "Email 2 Sent" to skip Email 1 and 2
- Set to "Email 3 Sent" to skip all emails

TIP 5: Batch Add Leads
-----------------------
You can add multiple leads at once
Each will get Email 1 automatically
