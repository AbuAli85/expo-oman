WHAT YOU NEED

FOR EMAIL 1 AUTOMATION:

CORRECT SETUP: ONE SET VARIABLES MODULE
================================================================================

FLOW:
-----

    ↓

MODULE 8 CONFIGURATION:
------------------------
ONE module with 3 variables:

Variable 1: name
   Value: {{1.0}}

Variable 2: email

Variable 3: language
   Value: {{1.3}}

That's it! Just ONE module.

================================================================================
WHEN TO USE HTTP MODULES
================================================================================

HTTP modules are for:
- Making API calls to external services
- Fetching data from web APIs
- Sending data to webhooks
- Custom integrations

NOT NEEDED FOR:
---------------
- Simple data organization (use Set Variables)
- Email automation (Resend module handles it)
- Google Sheets (Google Sheets module handles it)

================================================================================
WHEN TO USE MULTIPLE SET VARIABLES
================================================================================

Multiple Set Variables modules are for:
- Different scopes (one cycle vs execution)
- Organizing complex data flows
- Separating concerns in large scenarios

NOT NEEDED FOR:
---------------
- Simple email automation (one module is enough)
- Basic data organization (one module works fine)

================================================================================
YOUR SCENARIO: SIMPLE AND CLEAN
================================================================================

CURRENT FLOW (Without Module 8):
----------------------------------
Module 1 → Module 2 → Routes

RECOMMENDED FLOW (With Module 8):
----------------------------------
Module 1 → Module 8 → Module 2 → Routes

Module 8 stores:
- name
- email
- language

Then use {{8.email}} everywhere.

SIMPLE, CLEAN, EFFICIENT! ✅
