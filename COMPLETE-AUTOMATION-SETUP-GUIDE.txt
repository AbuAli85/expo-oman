================================================================================
TROUBLESHOOTING
================================================================================

ISSUE 1: Email 1 Not Sending
-----------------------------
Check:
- Is Scenario 1 turned ON?
- Is Google Sheets trigger configured correctly?
- Does the new row have valid email address?
- Check execution logs for errors

ISSUE 2: Email 2 Not Sending
-----------------------------
Check:
- Is Scenario 2 turned ON?
- Is schedule trigger set correctly?
- Are there leads with Status = "Email 1 Sent"?
- Is Email 1 Date more than 3 days ago?
- Check date filter is working

ISSUE 3: Email 3 Not Sending
-----------------------------
Check:
- Is Scenario 3 turned ON?
- Is schedule trigger set correctly?
- Are there leads with Status = "Email 2 Sent"?
- Check execution logs

ISSUE 4: Wrong Language Email Sent
-----------------------------------
Check:
- Router filters are set correctly
- Language values in Google Sheets match filters
- Case sensitivity (en vs EN, ar vs AR)

ISSUE 5: Duplicate Emails
--------------------------
Check:
- Status is being updated correctly
- Filters are working properly
- No duplicate rows in Google Sheets
