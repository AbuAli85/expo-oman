================================================================================
IMMEDIATE FIX FOR IML PARSING ERROR
================================================================================

PROBLEM:
--------
Modules 9 and 12 have IML parsing errors because of complex if() statements
with very long HTML strings. Make.com cannot parse these.

QUICK FIX (5 minutes):
----------------------

You need to add routers after the aggregators (Module 13 and Module 14) to
split by attendance response, then add separate email modules for each case.

================================================================================
FIX FOR MODULE 9 (In Route 1 - Update Existing Row)
================================================================================

STEP 1: Delete Module 9
-----------------------
1. Click on Module 9 (Resend → Send Email)
2. Click the trash icon
3. Confirm deletion

STEP 2: Add Router After Module 13
-----------------------------------
1. Click on Module 13 (Basic Aggregator)
2. Click "+" to add new module
3. Search for "Router" or "Flow Control → Router"
4. Add it - this becomes your new Module 9

STEP 3: Configure Router - Route 1A (Yes)
------------------------------------------
1. In Module 9 (Router), click "Add route"
2. Name: "Yes - Agreement"
3. Click "Set up a filter"
4. Add condition:
   - Field: {{2.attendance}}
   - Operator: equals
   - Value: "Yes, I'll attend"
5. Save route

STEP 4: Add Agreement Email in Route 1A
----------------------------------------
1. Inside Route 1A, click "+" to add module
2. Search for "Resend → Send an Email"
3. Configure:
   - From: sales@thesmartpro.io
   - To: [{"value": "{{2.email}}"}]
   - Subject: "Agreement - UAE Property Expo"
   - Content Type: HTML
   - HTML: Copy from AGREEMENT-EMAIL-TEMPLATE.html
     Replace {{4.name}} with {{2.name}}

STEP 5: Configure Router - Route 1B (No)
------------------------------------------
1. In Module 9 (Router), click "Add route"
2. Name: "No - Thank You"
3. Click "Set up a filter"
4. Add condition:
   - Field: {{2.attendance}}
   - Operator: equals
   - Value: "No, I cannot attend"
5. Save route

STEP 6: Add Thank You Email in Route 1B
----------------------------------------
1. Inside Route 1B, click "+" to add module
2. Search for "Resend → Send an Email"
3. Configure:
   - From: sales@thesmartpro.io
   - To: [{"value": "{{2.email}}"}]
   - Subject: "Thank You - UAE Property Expo"
   - Content Type: HTML
   - HTML: Copy from THANK-YOU-EMAIL-TEMPLATE.html
     Replace {{4.name}} with {{2.name}}
