WORKFLOW OVERVIEW
================================================================================

This automation sends a 3-email sequence to new leads from Google Sheets:
1. Invitation Email (sent immediately)
2. Reminder Email (sent 3 days after invitation)
3. Final Call Email (sent 4 days after reminder)

Optional: WhatsApp notification can be added for each email send.

================================================================================
ERROR HANDLING & BEST PRACTICES
================================================================================

1. Add Error Handlers:
   - Wrap each email module in error handler
   - Log failures to a separate "Errors" sheet
   - Continue workflow even if one email fails

2. Duplicate Prevention:
   - Check if email already sent before sending
   - Use Status column to track progress

3. Unsubscribe Handling:
   - Add filter to check if lead unsubscribed
   - Skip email sends for unsubscribed leads

4. Testing:
   - Test with a single row first
   - Verify email delivery and formatting
   - Check RTL rendering for Arabic emails

5. Rate Limiting:
   - If sending many emails, add delays between sends
