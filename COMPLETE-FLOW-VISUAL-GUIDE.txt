================================================================================
HOW TO CHECK CONNECTIONS IN MAKE.COM
================================================================================

METHOD 1: Visual Check
----------------------
1. Open scenario
2. Look at the flow diagram
3. You should see lines connecting modules
4. If no lines, modules are NOT connected

METHOD 2: Module Settings
--------------------------
1. Click on a module (e.g., Module 8)
2. Look for "Input" or "Connected to" field
3. It should show the previous module name
4. If empty or wrong, connection is broken

METHOD 3: Execution Log
------------------------
1. Run scenario
2. Check execution history
3. If Module 8 doesn't appear, it's not connected
4. If Module 8 appears but shows error, configuration issue

================================================================================
HOW TO FIX BROKEN CONNECTIONS
================================================================================

IF MODULE 8 NOT CONNECTED TO MODULE 1:
--------------------------------------
1. Click on Module 8
2. Look for connection/input option
3. Select "Module 1" or "Google Sheets Watch Rows"
4. Save

OR:

1. Hover over Module 1
2. Find output connector (small circle/dot)
3. Drag to Module 8 input connector
4. Release
5. Save

IF MODULE 2 NOT CONNECTED TO MODULE 8:
--------------------------------------
1. Click on Module 2 (Router)
2. Check input connection
3. Should be connected to Module 8
4. If not, connect it
5. Save

IF ROUTER ROUTES NOT CONNECTED:
-------------------------------
1. Click on Module 2 (Router)
2. Click on Route 1
3. Module 3 should be inside Route 1
4. If not, drag Module 3 into Route 1
5. Do same for Route 2 and Module 4
6. Save
