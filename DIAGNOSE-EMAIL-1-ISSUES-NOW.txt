COMMON FIXES
================================================================================

FIX 1: Webhook Not Receiving Data
───────────────────────────────────

If webhook trigger:
  □ Verify webhook URL in Google Apps Script
  □ Check Apps Script is running on form submit
  □ Test webhook manually
  □ Check Make.com webhook is active

FIX 2: Watch New Rows Broken
──────────────────────────────

If Watch New Rows:
  □ Right-click module → "Choose where to start"
  □ Re-select the sheet and range
  □ Check if rows/columns were deleted
  □ May need to recreate trigger

FIX 3: Module Reference Error
───────────────────────────────

If "Module references non-existing module":
  □ Check module numbers match
  □ Update references to correct module IDs
  □ Re-map variables in Set Variables module

FIX 4: Email "To" Field Error
───────────────────────────────

If 422 error in Resend:
  □ Change "To" field from array to string
  □ FROM: ["{{email}}"]
  □ TO: "{{email}}"

FIX 5: Variables Null
──────────────────────

If variables are null:
  □ Check form field IDs in Module 2
  □ Use Make.com field selector to map correctly
  □ Verify data structure matches expected format
