COMMON ISSUES
================================================================================

ISSUE 1: Form Options Don't Match Filter Values
─────────────────────────────────────────────────
Form has: "Yes, I will attend"
Filter checks: "Yes, I'll attend"
→ Mismatch! Won't match.

Fix: Update filters to match exact form values OR normalize in Module 2.

ISSUE 2: Extra Spaces
──────────────────────
Form value: "Yes, I'll attend " (trailing space)
Filter checks: "Yes, I'll attend" (no space)
→ Won't match due to trailing space.

Fix: Trim the value in Module 2:
  attendance: {{trim(1.answers.677abd25.textAnswers.answers[0].value)}}

ISSUE 3: Case Sensitivity
──────────────────────────
Form value: "YES, I'LL ATTEND" (uppercase)
Filter checks: "Yes, I'll attend" (mixed case)
→ Won't match if case-sensitive.

Fix: Make filters case-insensitive OR normalize to lowercase in Module 2.

ISSUE 4: Empty Value
─────────────────────
Form value: (empty or null)
Filter checks: "Yes, I'll attend"
→ Won't match empty value.

Fix: Check if form question is required, verify mapping.
