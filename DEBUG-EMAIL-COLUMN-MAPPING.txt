================================================================================
COMMON ISSUES
================================================================================

ISSUE 1: Column Index Off by One
----------------------------------
Make.com might use 0-based or 1-based indexing:
- 0-based: A=0, B=1, C=2, D=3
- 1-based: A=1, B=2, C=3, D=4

If {{1.1}} is null, try {{1.2}} or {{1.0}}

ISSUE 2: Headers Included in Count
-----------------------------------
If "includesHeaders" is true, the first row might be headers:
- Row 1: Headers (Name, Email, Phone, Language)
- Row 2: Data (Fahad Alamri, chairman@..., etc.)

This shouldn't affect column indices, but worth checking.

ISSUE 3: Empty Email Field
--------------------------
Check if the Email column in Google Sheets actually has data.
Maybe it's in a different column?
