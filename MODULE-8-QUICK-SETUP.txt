QUICK STEPS (5 MINUTES)
================================================================================

STEP 1: Add Module (1 minute)
------------------------------
1. Open Make.com → "Email 1 - Invitation"
2. Click "+" → Search "Set Variables" → Add
3. Position it between Module 1 and Module 2

STEP 2: Connect (1 minute)
---------------------------
1. Connect Module 1 → Module 8
2. Connect Module 8 → Module 2

STEP 3: Configure (2 minutes)
------------------------------
Open Module 8, add 3 variables:

1. name = {{1.0}}
2. email = {{1.1}}
3. language = {{1.3}}

Scope: One cycle
Save

STEP 4: Update Modules (1 minute)
----------------------------------
Module 3:
- Filter: {{8.language}} = "en"
- To: [{"value": "{{8.email}}"}]
- Subject: "{{8.name}}, You're Invited..."

Module 4:
- Filter: {{8.language}} = "ar"
- To: [{"value": "{{8.email}}"}]
- Subject: "{{8.name}}، أنت مدعو!..."

Save all
