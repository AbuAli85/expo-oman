================================================================================
ADD ACCEPT/REJECT WORKFLOW TO EMAIL
Client Form with Accept/Reject Links
================================================================================

You want to add:
1. Link in email to accept or reject
2. Form/page for client to respond
3. If accept: continue to agreement
4. If reject: handle differently

================================================================================
OPTION 1: SIMPLE - USE GOOGLE FORMS (EASIEST)
================================================================================

STEP 1: Create Google Form
---------------------------
1. Go to Google Forms
2. Create new form: "UAE Property Expo - Response"
3. Add question:
   - "Will you attend the expo?"
   - Type: Multiple choice
   - Options: "Yes, I'll attend" / "No, I cannot attend"
4. Add fields:
   - Name (pre-filled from email)
   - Email (pre-filled from email)
   - Additional comments (optional)
5. Get form link

STEP 2: Add Link to Email
--------------------------
In Module 3 (English Email) and Module 4 (Arabic Email):

Update the "Register Now" button link:
- FROM: href="#"
- TO: href="YOUR_GOOGLE_FORM_LINK?entry.XXXXX={{1.0}}&entry.YYYYY={{1.1}}"

Replace:
- YOUR_GOOGLE_FORM_LINK = your form URL
- entry.XXXXX = field ID for Name
- entry.YYYYY = field ID for Email

STEP 3: Create Response Handler Scenario
----------------------------------------
Create NEW scenario: "Handle Expo Response"

Trigger: Google Forms → Watch Responses
Action: Check response
- If "Yes" → Send agreement email
- If "No" → Send thank you email

STEP 4: Update Google Sheets
----------------------------
Add new column: "Response" (Column K)
Update based on form response

================================================================================
OPTION 2: ADVANCED - CUSTOM WEBPAGE (MORE CONTROL)
================================================================================

STEP 1: Create Landing Page
----------------------------
Create a webpage with:
- Accept button → Links to agreement form
- Reject button → Links to feedback form
- Personalized with client name

STEP 2: Add Tracking to Links
------------------------------
In email, use UTM parameters:
- Accept link: https://yoursite.com/accept?email={{8.email}}&token=UNIQUE_TOKEN
- Reject link: https://yoursite.com/reject?email={{8.email}}&token=UNIQUE_TOKEN

STEP 3: Create Webhook Handler
-------------------------------
Create NEW scenario: "Handle Accept/Reject"

Trigger: Webhooks → Custom webhook
- URL: https://hook.eu2.make.com/YOUR_WEBHOOK_ID
- Method: POST

STEP 4: Process Response
-------------------------
In webhook handler:
- Check response (accept/reject)
- Update Google Sheets
- Send appropriate email

---------------------------------
In "Handle Expo Response" scenario:

1. Webhook trigger receives:

CREATE RESPONSE HANDLER SCENARIO
================================================================================

SCENARIO: "Handle Expo Response"

MODULE 1: Webhooks → Custom webhook
------------------------------------
- Method: GET (for simple links)
- OR Method: POST (for forms)

MODULE 2: Router
-----------------
Route 1 Filter: {{query.response}} = "accept"
Route 2 Filter: {{query.response}} = "reject"

ROUTE 1 (ACCEPT):
-----------------
Module 3: Google Sheets → Update Row
   - Find row by email: {{query.email}}
   - Update:
     * Response (Column K): "Accepted"
     * Response Date (Column L): {{now}}

Module 4: Resend → Send Agreement Email
   - To: {{query.email}}
   - Subject: "Agreement - UAE Property Expo"
   - Include agreement details/form

OR

Module 4: HTTP → Make a Request
   - Redirect to agreement form
   - URL: https://yoursite.com/agreement?email={{query.email}}

ROUTE 2 (REJECT):
------------------
Module 5: Google Sheets → Update Row
   - Find row by email: {{query.email}}
   - Update:
     * Response (Column K): "Rejected"
     * Response Date (Column L): {{now}}

Module 6: Resend → Send Thank You Email
   - To: {{query.email}}
   - Subject: "Thank You - UAE Property Expo"
   - Thank them for interest

================================================================================
UPDATE GOOGLE SHEETS STRUCTURE
================================================================================

Add new columns:

Column K: Response
   - Values: "Accepted", "Rejected", "Pending"

Column L: Response Date
   - Date when response received

Column M: Agreement Status (if accepted)
   - Values: "Pending", "Sent", "Signed"
